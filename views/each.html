Chaining

The each() method specifies a function to run for each matched element. Tip: return false can be used to stop the loop early.



Syntax:

$(selector).each(function(index,element))

Example:

            script>
            $("button").click(function(){
                $("li").each(function(){
                  alert($(this).text())
                });
              });
            /script>
